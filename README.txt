Buenas, antes de comenzar se debe seguir una serie de pasos que permiten la ejecucion
de algunas funciones/codigos en los archivos Jupyter.

Paso 1: Se debe instalar la API de Steam. Para ello se coloca en la terminal: -py -m pip install steamapi

Paso 2 (opcional): Una vez instalada la API, debe obtener una Key desde su cuenta de steam. 

Paso 3 (opcional): Obtenida la key, dirigase a la siguiente pagina si desea ver los recursos que
ofrece la API: https://steamapi.xpaw.me/#

Los pasos opcionales son necesarios solo si desea trabajar por su cuenta con la API.
